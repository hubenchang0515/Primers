#命令行及其基本操作

Linux 上默认的命令行通常是 sh 或 bash，后者大致兼容前者。

绝大多数 Linux 发行版可以通过默认的 Ctrl + Alt + T 快捷键打开终端模拟器。或者找到类似下图的黑色图标点击打开，

如图所示 planc@raspberrypi:~ $ 是命令提示符，其中 planc 是用户名，是 主机名，~ 是当前工作目录。

在命令提示符后输入命令，然后按回车执行。以下是一些常用命令：

ls 目录                     # 列出目录内容，省略 <目录> 则列出当前工作目录
cd 目录                     # 将工作目录切换到 <目录>
rm 文件                     # 删除 <文件>
cp 文件1 文件2              # 将 <文件1> 复制到 <文件2>
mv 文件1 文件2              # 将 <文件1> 移动到 <文件2>
echo 文本                   # 显示 <文本>
echo $变量                  # 显示 <变量> 的值

help                        # 查看所有内建命令
man 命令                    # 查看 <命令> 的手册

示例:

# 此虚拟环境只能运行内建命令
help    # 查看所有内建命令

运行可执行文件时，要求该文件在 PATH 环境变量包含的目录中。如果要运行当前目录的文件，则需要输入 ./文件名。

echo $PATH                              # 查看 PATH 环境变量
export PATH="路径1:路径2:...:路径n;"    # 修改 PATH 环境变量，使用英文冒号分隔路径
export PATH="$env:PATH:路径"            # 将 <路径> 添加到 PATH 环境变量中，使用英文冒号分隔路径
