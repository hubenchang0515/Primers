#CSS 的颜色

属性描述查看
color颜色查看
background-color背景颜色查看

颜色的表示方法：

  1. 颜色名，例如 red, pink, green 等
  2. #RGBA，其中 R 为红色通道，G 为绿色通道，B 为蓝色通道，A 为 Alpha 通道，取值均为十六进制 0 到 f，不区分大小写，下同
  3. #RRGGBBAA，每个通道两位
  4. #RGB，省略 alpha 通道
  5. #RRGGBB，省略 alpha 通道
  6. rgba(R, G, B)，其中 R 为红色通道，G 为绿色通道，B 为蓝色通道，取值为整数 0 到 255，下同
  7. rgba(R, G, B, A)，A 为 Alpha 通道取值为小数 0 到 1

Alpha 通道用于 颜色重叠时作为参数 参与颜色混合算法的计算，通常表示不透明度。

可以使用 颜色转换工具 进行格式转换。

#color

CSS 中使用 color 设置颜色，例如：

<p style="color:pink;">Primers —— 你的编程学习伙伴</p>
<p style="color:#39c5bb;">Primers —— 你的编程学习伙伴</p>
<p style="color:rgb(102, 204, 255);">Primers —— 你的编程学习伙伴</p>
